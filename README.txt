Si quieres que cada vez que el archivo se guarde se recargue la app cuando al tengas abierta
ejecuta este comando en la terminal de vscode o en una termianl dentro de la carpeta
watchmedo auto-restart --pattern="*.py" --recursive -- python Main/app.py

instalar antes dentro de la carpeta si lo haces desde tu pc si lo haces desde la nube, no sera necesario
pip install watchdog

usar normal
python Main/app.py

instalar dependencia pip install requirements.txt
